# CarDekho Sales Analysis using SQL

## Project Overview

This project demonstrates SQL querying skills by analyzing a used car dataset from CarDekho.

The objective was to answer business questions such as:

- Total number of cars
- Cars available in a particular year
- Fuel-wise car count
- Year-wise sales analysis
- Cars sold between specific years
- Finding years with more than 100 cars

---

## Database
MySQL

---

## Skills Used

- SELECT
- WHERE
- GROUP BY
- HAVING
- ORDER BY
- Aggregate Functions
- COUNT()
- Filtering
- Database Creation

---

## Business Questions Solved

1. Count total cars.
2. Find cars available in 2023.
3. Find cars available in 2020, 2021, and 2022.
4. Count cars by year.
5. Count diesel cars in 2020.
6. Count petrol cars in 2020.
7. Fuel-wise yearly car count.
8. Find years having more than 100 cars.
9. Count cars sold between 2015 and 2023.
10. Display all cars sold between 2015 and 2023.
