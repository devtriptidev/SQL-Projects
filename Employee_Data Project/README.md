# Employee Database SQL Analysis

## Project Overview

This project demonstrates comprehensive SQL skills by solving real-world business problems using the MySQL **Employees** sample database.

The project covers everything from basic SQL queries to advanced concepts such as joins, subqueries, views, stored procedures, and data manipulation (CRUD operations). It serves as a hands-on portfolio project showcasing SQL proficiency required for Data Analyst, Business Analyst, and SQL Developer roles.

---

## Tech Stack

- MySQL
- SQL
- MySQL Workbench

## Database

This project uses the **MySQL Employees Sample Database**, which contains information about:

- Employees
- Departments
- Salaries
- Department Managers
- Employee Titles
- Department Assignments

## Objectives

The goal of this project is to practice and demonstrate SQL by solving business-oriented problems such as:

- Employee data retrieval
- Salary analysis
- Department-wise reporting
- Manager identification
- Employee hiring trends
- Data manipulation (INSERT, UPDATE, DELETE)
- Database object creation
- Performance of joins and subqueries

## Business Problems Solved

Some of the business questions solved include:

- Retrieve employees hired after a specific date.
- Find employees based on gender and hiring date.
- Count employees and managers.
- Analyze salary distribution.
- Identify employees with multiple department contracts.
- Calculate average salary by department.
- Find average salary by gender.
- List all managers with department information.
- Create duplicate tables for testing.
- Remove duplicate records.
- Build views for reporting.
- Create reusable stored procedures.
- Perform CRUD operations on employee records.
